Adds 'Share This Story' buttons to the end of a single post before the comments. Posts can be shared on Twitter, Facebook and Google+.

== Description ==

Designed for use with [The J A Mortram WordPress Theme](http://wordpress.org/themes/the-j-a-mortram). 

Once activated, plugin adds 'Share This Story' buttons to the end of a single post before the comments. Posts can be shared on Twitter, Facebook and Google+. This plugin does not embed tags from any of the above services in your WordPress site. Instead it simply posts the page URL and any other information relevant to each network's API.

== Installation ==

= For an automatic installation through WordPress =

* Go to the 'Add New' plugins screen in your WordPress admin area
* Search for 'The J A Mortram Share This Story'
* Click 'Install Now' and activate the plugin
* 'Share This Story' buttons are added to single post pages once the plugin is activated

= For a manual installation via FTP =

* Upload the folder you download to the directory `/wp-content/plugins/`
* Activate the plugin through the 'Plugins' screen in your WordPress admin area

== Frequently Asked Questions ==

= Where are the settings? =

There are no settings, just activate the plugin to display the share buttons.
